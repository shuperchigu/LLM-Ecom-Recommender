# ­Ъџђ Generative AI-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃюрЃў рЃАрЃљрЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ

### **рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ (Executive Summary)**

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃ░рЃўрЃЉрЃарЃўрЃЊрЃБрЃџ рЃАрЃљрЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА **PostgreSQL-рЃўрЃА (рЃарЃћрЃџрЃљрЃфрЃўрЃБрЃарЃў рЃџрЃЮрЃњрЃўрЃЎрЃљ)** рЃЊрЃљ **Google Gemini API**-рЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃА. рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃў рЃцрЃЮрЃЎрЃБрЃАрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ **рЃАрЃЎрЃћрЃўрЃџрЃўрЃарЃћрЃЉрЃљрЃАрЃљ** рЃЊрЃљ **рЃАрЃўрЃАрЃгрЃарЃљрЃцрЃћрЃќрЃћ**, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА:

* **рЃерЃћрЃЊрЃћрЃњрЃў:** рЃбрЃћрЃАрЃбрЃћрЃЉрЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ **10%-рЃўрЃљрЃюрЃў рЃќрЃарЃЊрЃўрЃА рЃърЃЮрЃбрЃћрЃюрЃфрЃўрЃљрЃџрЃў** рЃЎрЃЮрЃюрЃЋрЃћрЃарЃАрЃўрЃўрЃА рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃћрЃџрЃерЃў.
* **рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃљ:** рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃБрЃџрЃў ML-рЃўрЃАрЃњрЃљрЃю рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃўрЃЌ, рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА **рЃАрЃгрЃарЃљрЃц A/B рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљрЃА** рЃЊрЃљ **рЃЏрЃљрЃарЃбрЃўрЃЋ рЃљрЃЊрЃљрЃърЃбрЃљрЃфрЃўрЃљрЃА** рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

---

## ­ЪњА рЃАрЃљрЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃЮ рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃАрЃўрЃдрЃарЃЏрЃћ (The Ranking Engine)

рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃўрЃюрЃЮрЃЋрЃљрЃфрЃўрЃљрЃљ рЃўрЃА, рЃарЃЮрЃЏ рЃўрЃА рЃљрЃа рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ рЃћрЃарЃЌ рЃЏрЃЮрЃЊрЃћрЃџрЃА. рЃарЃћрЃџрЃћрЃЋрЃљрЃюрЃбрЃБрЃарЃЮрЃЉрЃљ рЃЏрЃўрЃўрЃдрЃгрЃћрЃЋрЃљ рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў, рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃюрЃў (Multi-Layered) рЃцрЃљрЃЦрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ:

### I. рЃўрЃюрЃбрЃћрЃарЃћрЃАрЃўрЃА рЃгрЃЮрЃюрЃўрЃЌрЃў рЃњрЃљрЃљрЃюрЃњрЃљрЃарЃўрЃерЃћрЃЉрЃљ (Decay Factor)
* **рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃў:** рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЉрЃЮрЃџрЃЮ 500 рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ.
* **рЃгрЃЮрЃюрЃљ:** рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃћрЃЉрЃў рЃцрЃљрЃАрЃЊрЃћрЃЉрЃљ рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (рЃДрЃўрЃЊрЃЋрЃљ > рЃЎрЃљрЃџрЃљрЃЌрЃљрЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ > рЃюрЃљрЃ«рЃЋрЃљ) рЃЊрЃљ рЃћрЃЦрЃЋрЃћрЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЉрЃљ **рЃЊрЃарЃЮрЃерЃў рЃЊрЃљрЃерЃџрЃўрЃА рЃцрЃљрЃЦрЃбрЃЮрЃарЃА (Time-Decay)**, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃћрЃЉрЃў рЃўрЃДрЃЮрЃА рЃБрЃцрЃарЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў.

### II. рЃЎрЃљрЃюрЃЊрЃўрЃЊрЃљрЃбрЃћрЃЉрЃўрЃА рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ рЃЊрЃљ рЃарЃљрЃюрЃЪрЃўрЃарЃћрЃЉрЃљ (SQL-Engine)
рЃЎрЃљрЃюрЃЊрЃўрЃЊрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃарЃЕрЃћрЃЋрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃБрЃарЃљрЃЊ рЃЊрЃљ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ, рЃарЃЌрЃБрЃџрЃў SQL рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃфрЃљрЃЋрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃА:

* **рЃЉрЃарЃћрЃюрЃЊрЃўрЃА рЃџрЃЮрЃўрЃљрЃџрЃЮрЃЉрЃљ:** рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃбрЃЮрЃъ 3 рЃџрЃЮрЃўрЃљрЃџрЃБрЃа рЃЉрЃарЃћрЃюрЃЊрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЦрЃЋрЃћрЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљрЃерЃў, рЃўрЃдрЃћрЃЉрЃћрЃю рЃБрЃЏрЃљрЃдрЃџрЃћрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃА.
* **рЃцрЃљрЃАрЃўрЃА рЃърЃарЃћрЃцрЃћрЃарЃћрЃюрЃфрЃўрЃћрЃЉрЃў:** рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў рЃџрЃљрЃњрЃЊрЃћрЃЉрЃљ рЃўрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, рЃћрЃбрЃћрЃЋрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃљрЃЌрЃў рЃцрЃљрЃАрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЎрЃЮрЃЏрЃцрЃЮрЃарЃбрЃБрЃџ рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃерЃў.
* **рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃАрЃњрЃљрЃЋрЃАрЃћрЃЉрЃљ:** `pgvector`-рЃўрЃА embedding-рЃћрЃЉрЃўрЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

### III. рЃЏрЃарЃљрЃЋрЃљрЃџрЃцрЃћрЃарЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃЊрЃљ рЃАрЃџрЃЮрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ
* **рЃГрЃћрЃарЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў:** рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ 30 рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљрЃерЃў, рЃљрЃарЃфрЃћрЃарЃЌ рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљрЃА рЃљрЃа рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЊрЃљрЃўрЃЎрЃљрЃЋрЃЮрЃА 10 рЃАрЃџрЃЮрЃбрЃќрЃћ (33%) рЃЏрЃћрЃбрЃў, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃцрЃћрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљрЃА.
* **рЃўрЃюрЃбрЃћрЃарЃџрЃўрЃЋрЃўрЃюрЃњрЃў (Interleaving):** Gemini-рЃА рЃЏрЃўрЃћрЃа рЃарЃћрЃарЃљрЃюрЃЪрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃћрЃЉрЃў рЃћрЃарЃЌрЃўрЃљрЃюрЃЊрЃћрЃЉрЃљ рЃљрЃЏ рЃГрЃћрЃарЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃА рЃЊрЃљрЃфрЃЋрЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃўрЃЦрЃюрЃљрЃА рЃљрЃфрЃўрЃџрЃћрЃЉрЃБрЃџрЃў рЃћрЃарЃЌрЃў рЃЊрЃЮрЃЏрЃўрЃюрЃљрЃюрЃбрЃў рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃЮрЃюрЃљ.

---

### **BIA (Buy It Again) рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ**

рЃћрЃА рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃў рЃљрЃАрЃћрЃЋрЃћ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџ **"Buy It Again"** рЃЏрЃЮрЃЊрЃБрЃџрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **рЃгрЃћрЃАрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџ рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА** рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃўрЃЌрЃў рЃерЃћрЃАрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:

* **рЃЏрЃўрЃќрЃљрЃюрЃў:** рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃљ **рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃА рЃфрЃўрЃЎрЃџрЃўрЃА** (Replenishment Cycle) рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ.
* **рЃџрЃЮрЃњрЃўрЃЎрЃљ:** рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃгрЃћрЃАрЃћрЃЉрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃА рЃЊрЃљ рЃЉрЃЮрЃџрЃЮ рЃерЃћрЃАрЃДрЃўрЃЊрЃЋрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃА, рЃарЃљрЃЌрЃљ рЃўрЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃЏрЃћрЃбрЃДрЃЋрЃћрЃџрЃЮрЃА **рЃерЃћрЃЋрЃАрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў** (`predicted_replenish_date`).
* **рЃбрЃћрЃЦрЃюрЃўрЃЎрЃљ:** рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **ThreadPoolExecutor**-рЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## ­ЪДа рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃ«рЃћрЃЊрЃЋрЃљ (CDO's Strategy)

рЃЕрЃЋрЃћрЃюрЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃўрЃЏрЃФрЃџрЃљрЃЋрЃарЃћрЃА** рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, рЃАрЃљрЃЊрЃљрЃф рЃћрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ, рЃарЃљрЃф рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃќрЃарЃЊрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљрЃА.

| рЃћрЃбрЃљрЃърЃў | рЃцрЃЮрЃЎрЃБрЃАрЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ |
| :--- | :--- | :--- | :--- |
| **1. Smart Profiling** | **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃфрЃўрЃцрЃарЃБрЃџрЃў рЃљрЃюрЃљрЃЉрЃћрЃГрЃЊрЃў** | рЃљрЃљрЃюрЃљрЃџрЃўрЃќрЃћрЃЉрЃА рЃЉрЃЮрЃџрЃЮ 500 рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃљрЃА (рЃДрЃўрЃЊрЃЋрЃљ, рЃЎрЃљрЃџрЃљрЃЌрЃљ, рЃюрЃљрЃ«рЃЋрЃљ) рЃЊрЃарЃЮрЃерЃў рЃЊрЃљрЃерЃџрЃўрЃџрЃў рЃгрЃЮрЃюрЃўрЃЌ. рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЉрЃарЃћрЃюрЃЊрЃўрЃА рЃџрЃЮрЃўрЃљрЃџрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃцрЃљрЃАрЃўрЃА рЃърЃарЃћрЃцрЃћрЃарЃћрЃюрЃфрЃўрЃћрЃЉрЃА. | **Python (Pandas), PostgreSQL** |
| **2. Product Embedding** | **рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃБрЃарЃў рЃФрЃўрЃћрЃЉрЃўрЃА рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃў** | рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА (рЃарЃћрЃўрЃбрЃўрЃюрЃњрЃў, рЃцрЃљрЃАрЃў, рЃЉрЃарЃћрЃюрЃЊрЃў) рЃЋрЃћрЃЦрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ Gemini-рЃўрЃЌ. | **Gemini API, PostgreSQL + pgvector** |
| **3. Candidate Generation** | **рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ рЃЊрЃљ рЃарЃљрЃюрЃЪрЃўрЃарЃћрЃЉрЃљ SQL-рЃерЃў** | рЃЎрЃљрЃюрЃЊрЃўрЃЊрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃарЃЕрЃћрЃЋрЃљ рЃ«рЃЊрЃћрЃЉрЃљ **рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа SQL-рЃерЃў** рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃюрЃў `ORDER BY` рЃџрЃЮрЃњрЃўрЃЎрЃўрЃЌ. рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА AI-рЃќрЃћ рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА. | **SQL (Multi-Layered), pgvector** |
| **4. Generative Reranking** | **рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃќрЃБрЃАрЃбрЃћ** | рЃерЃћрЃарЃЕрЃћрЃБрЃџрЃў рЃЎрЃљрЃюрЃЊрЃўрЃЊрЃљрЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃљ Gemini-рЃўрЃА рЃЏрЃўрЃћрЃа, "рЃЊрЃљрЃДрЃљрЃЋрЃў рЃЊрЃљ рЃўрЃЉрЃљрЃбрЃЮрЃюрЃћ" рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃўрЃЌ. рЃћрЃА рЃќрЃарЃЊрЃўрЃА рЃарЃћрЃџрЃћрЃЋрЃљрЃюрЃбрЃБрЃарЃЮрЃЉрЃљрЃА. | **Google Gemini API (LLM)** |
| **5. Storage & Analytics** | **A/B рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃќрЃљрЃЮрЃЉрЃљ** | рЃАрЃарЃБрЃџрЃў рЃАрЃћрЃАрЃўрЃљ рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃарЃћрЃџрЃљрЃфрЃўрЃБрЃа рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃерЃў рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА. | **PostgreSQL** |

## РџЎ№ИЈ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃАрЃбрЃћрЃЎрЃў рЃЊрЃљ рЃўрЃюрЃЪрЃўрЃюрЃарЃБрЃџрЃў рЃАрЃўрЃЏрЃДрЃљрЃарЃћ

* **Generative AI:** Google Gemini API (LLM)
* **Data Engineering:** Apache Airflow (ETL Orchestration), Python (Multithreading - `ThreadPoolExecutor`)
* **Data Persistence:** PostgreSQL (Connection Pooling), `pgvector`
* **Infrastructure:** AWS (EC2)
* **Robustness:** **Exponential Backoff Retry Logic** (API-рЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃФрЃџрЃћрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА).

## ­Ъџђ рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ (End-to-End Guide)

### 1\. рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

1.  **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў:** рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ `requirements.txt` (рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА `google-generativeai`, `pgvector`, `pandas`).
    ```bash
    pip install -r requirements.txt
    ```
2.  **API/DB рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ:** рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ Gemini API Key рЃЊрЃљ PostgreSQL рЃЎрЃарЃћрЃЊрЃћрЃюрЃфрЃўрЃљрЃџрЃћрЃЉрЃў.
3.  **рЃЉрЃљрЃќрЃўрЃА рЃАрЃЦрЃћрЃЏрЃљ:** рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃАрЃЦрЃћрЃЏрЃћрЃЉрЃў PostgreSQL-рЃќрЃћ.

### 2\. рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў

* **рЃћрЃЏрЃЉрЃћрЃЊрЃўрЃюрЃњрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ:**
    ```bash
    python3 generate_embeddings.py
    ```
* **рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ:**
    ```bash
    python3 generate_profiles.py
    ```
* **рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ (рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ):**
    ```bash
    python3 recommender.py
    ```
